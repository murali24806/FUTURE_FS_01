# Murali Portfolio

This is my personal portfolio website where you can learn more about me, my skills, projects, and experience.
It showcases my work in web development and technology.

## 🚀 Features

* About Me section
* Skills and Technologies
* Projects Showcase
* Contact Information

## 📦 Installation

Clone the repository:

git clone https://github.com/murali24806/FUTURE_FS_01.git

Navigate to the project folder:

cd FUTURE_FS_01

Install dependencies:

npm install

## ▶️ Run the Project

To start the development server:

npm start

## 🌐 View the Website

After running the project, open your browser and visit:

http://localhost:3000
